# Agile/Scrum Preset

**Nombre**: agile-scrum  
**Versión**: 0.1.0

Integra las ceremonias y artefactos de Scrum dentro del flujo Spec-Kit.

## Comandos incluidos
- `/pm.sprint-plan`
- `/pm.retrospective`
- `/pm.daily-standup`
- `/pm.backlog-grooming`

## Uso
```bash
specify preset add agile-scrum
```
