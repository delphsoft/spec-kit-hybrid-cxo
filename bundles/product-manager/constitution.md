# Constitution - Product Manager Bundle (AI Agent Oriented)

## Principios Generales
1. **Agent-First Development**: Todo feature debe ser desglosado y asignado a agentes especializados.
2. **Spec as Single Source of Truth**: Las especificaciones deben ser completas, claras y revisadas antes de implementación.
3. **Continuous Validation**: Usar simulaciones y métricas en cada etapa.
4. **Cross-Functional Alignment**: Toda spec debe considerar input de CTO, CFO, Legal y Marketing.
5. **Iterative + Measurable**: Cada feature debe tener OKRs claros y métricas de éxito.

## Guardrails
- Nunca implementar sin spec revisada.
- Todo cambio debe actualizar la spec correspondiente.
- Priorizar Value / Effort / Risk en todas las decisiones.
- Mantener trazabilidad completa (Spec → Plan → Tasks → Implementation).
- Validar privacy, costos y experiencia de usuario en cada feature.
