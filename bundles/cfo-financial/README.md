# CFO Financial Bundle

**Rol:** Chief Financial Officer / Financial Strategist

**Enfoque:** Orientado a desarrollo con agentes IA. Integra análisis financiero, modelado de costos, unit economics y ROI directamente en el flujo de Spec-Driven Development.

## Comandos principales

- `/cfo.financial-model` — Genera modelo financiero para una feature o proyecto
- `/cfo.cost-structure` — Análisis detallado de costos (fixed, variable, cloud, etc.)
- `/cfo.ROI-analysis` — Cálculo de retorno de inversión y payback period
- `/cfo.budget-plan` — Planificación presupuestaria y forecasting
- `/cfo.unit-economics` — Modelado de LTV, CAC, margins, etc.
- `/cfo.risk-financial` — Análisis de riesgos financieros

## Integración con otros bundles
- Trabaja con **Product Manager** para priorizar features por valor/costo
- Colabora con **CTO** en trade-offs costo vs performance
- Se alimenta de **Stage Adapter** (más conservador en startup-mvp)

## Uso recomendado
```bash
/pm.feature-breakdown
/cfo.financial-model
/cto.architecture-review
/legal.risk-assessment
```
