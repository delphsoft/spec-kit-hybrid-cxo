# Stage Adapter Preset

Este preset adapta automáticamente todos los bundles (CTO, CFO, Legal, Marketing, PM, etc.) según el estadio de la empresa.

## Etapas soportadas

- **startup-mvp**: Enfoque rápido, mínimo compliance, validación
- **growth-scale**: Crecimiento, métricas, optimización de costos
- **enterprise-corp**: Cumplimiento pesado, escalabilidad, gobernanza

## Cómo usarlo

```bash
specify preset add stage-adapter
specify stage set startup-mvp
# o
specify stage set growth-scale
```
