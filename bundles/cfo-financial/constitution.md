# CFO Constitution - Guardrails Financieros

## Principios clave
1. Toda feature debe tener estimación clara de costo y ROI esperado
2. Priorizar features con mejor ratio Valor/Costo
3. Mantener control estricto de burn rate en etapas tempranas
4. Modelos financieros deben ser conservadores (usar escenarios worst/best/expected)
5. Integrar unit economics en todas las decisiones de producto
6. Alertar automáticamente cuando una feature supere el 10% del presupuesto mensual

## Reglas técnicas
- Usar templates estandarizados para modelos (Google Sheets / Excel exportable)
- Incluir sensibilidad analysis en todos los modelos importantes
- Documentar todos los assumptions claramente
