# Suite Completa de Análisis Financiero (CFO Bundle)

Esta suite integra todas las herramientas de análisis financiero para uso con agentes IA.

## Herramientas Incluidas

### 1. Unit Economics
- CAC, LTV, LTV:CAC Ratio
- Payback Period
- Contribution Margin

### 2. Feature / Project Financials
- Break-Even Analysis
- Payback Period Analysis
- ROI & NPV

### 3. Advanced Simulation
- Sensitivity Analysis
- **Monte Carlo Simulation** (con gráficos de distribución)
- Scenario Planning

### 4. Decision Support
- Weighted Decision Matrix
- Trade-off Tables

## Comandos Principales

- `/cfo.unit-economics`
- `/cfo.break-even`
- `/cfo.payback-period`
- `/cfo.monte-carlo --iterations 10000 --with-charts`
- `/cfo.sensitivity-analysis`
- `/pm.weighted-decision`

## Integración Recomendada
Usar dentro de `/pm.orchestrate` para features importantes.
