# COO Operations Bundle

**Rol**: Chief Operating Officer - Orquestador de Operaciones, Automatización y Eficiencia Escalable.

Este bundle está diseñado para trabajar con **agentes IA** en entornos de desarrollo Spec-Driven.

## Principios Clave
- Operations as Code + Agents
- Automatización inteligente primero
- Métricas operativas medibles
- Coordinación entre todos los C-Level bundles

## Comandos Principales

- `/coo.operations-review`
- `/coo.process-automation`
- `/coo.scaling-plan`
- `/coo.kpi-dashboard`
- `/coo.team-structure` (Human + AI Agents)
- `/coo.efficiency-audit`
- `/coo.risk-operational`

## Integración
Trabaja fuertemente con Stage Adapter y todos los demás bundles (PM, CTO, CFO, etc.).
