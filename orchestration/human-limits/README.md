# Human Limits in Multi-Agent Orchestration

Inspired by [Addy Osmani - Cognitive Parallel Agents](https://addyosmani.com/blog/cognitive-parallel-agents/)

This section addresses the **human cognitive constraints** when orchestrating multiple AI agents.

## Core Concepts

- **Personal Ceiling**: Maximum number of parallel agents a human can effectively manage.
- **Context Switching Cost**
- **Comprehension Debt**
- **Trust Calibration**
- **Background Anxiety Tax**

## Key Guidelines

1. Know your personal limit (typically 3-7 agents depending on complexity).
2. Prefer tighter scope over more agents.
3. Use time-boxing and regular checkpoints.
4. Implement Trust Scores and Quality Gates.
