# Personal Ceiling Calibration

## Comando
`/pm.calibrate-ceiling`

## How it works

The Product Manager Agent asks the human:
- Current complexity level
- Time available
- Experience with parallel agents

Then recommends:
- Optimal number of parallel agents
- Suggested scope reduction
- Session time-box (e.g. 90 minutes)

## Example Output
"Recommended max: 4 agents for this feature. Reduce UIUX scope to focus only on core flows."
